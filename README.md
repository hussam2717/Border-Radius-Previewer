## Border-Radius-Previewer
A simple and interactive tool to visualize and generate CSS border-radius values in real time.

## Live Demo - https://hussam2717.github.io/Border-Radius-Previewer/

## ✨ Features
🎯 Real-time border-radius preview
🎚️ Interactive slider control
📋 One-click CSS copy button
💡 Clean and modern UI design

🛠️ Built With
HTML5
CSS3
JavaScript

📂 Project Structure
 ┣ 📄 index.html
 ┣ 📄 style.css
 ┣ 📄 script.js

 🎯 Use Case
This tool helps developers quickly test and apply border-radius styles without manually writing values.
